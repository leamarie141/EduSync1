# EduSync LMS Prototype

EduSync is a Learning Management System (LMS) prototype designed to help students manage notes, study timer, calendars, and track progress in one intuitive platform.  
This project includes both a **backend (Node.js + Express)** and a **frontend (HTML, CSS, JS)**.

---

## 🚀 Features
- **Landing Page**: Home, About, Features, Get Started
- **Authentication**: Sign Up, Sign In, Log Out
- **Dashboard**:
  - Notes: Create, edit, delete, share
  - Study Timer: Study, Long break, Short break
  - calendar: Calendar view, add events, export PDF
  - Progress: Tract Progress
  - Profile: Account settings, edit info, change password
